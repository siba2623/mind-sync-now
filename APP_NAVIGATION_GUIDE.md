## 🎯 User Flows

### Flow 1: Complete Mental Health Assessment
```
Dashboard → Health Hub → Screening Tab → Select PHQ-9 → 
Answer 9 Questions → View Results → (If severe) See Crisis Support
```

### Flow 2: Track Medication
```
Dashboard → Health Hub → Medication Tab → Add Medication → 
Set Reminder → Mark as Taken Daily
```

### Flow 3: Voice Mood Check
```
Dashboard → Health Hub → Voice Tab → Start Recording → 
Speak for 30 seconds → Stop → View AI Analysis
```

### Flow 4: Export Data for Doctor
```
Dashboard → Health Hub → Export Tab → Select Data → 
Choose Format → Download → Share with Provider
```

### Flow 5: Enroll in Wellness Program
```
Dashboard → Health Hub → Discovery Tab → Available Programs → 
Select Program → Enroll Now → Track Progress
```

## 🎨 Color Coding

### Status Badges:
- 🟢 **Green** - Completed, Taken, Minimal severity
- 🟡 **Yellow** - Time to take, Mild severity
- 🟠 **Orange** - Moderate severity
- 🔴 **Red** - Severe, Crisis, Emergency
- 🔵 **Blue** - Active, In progress
- ⚪ **Gray** - Paused, Inactive

### Card Highlights:
- **Red border** - Health Hub (priority feature)
- **Blue gradient** - Discovery Health
- **Purple gradient** - Mental health features
- **Green gradient** - Wellness activities
- **Yellow gradient** - Vitality points
