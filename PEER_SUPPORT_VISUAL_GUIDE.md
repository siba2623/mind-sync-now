## 📱 Mobile Gestures

### Swipe Actions
```
Message List:
← Swipe left  = Report
→ Swipe right = Reply (planned)

Group List:
← Swipe left  = Leave group
→ Swipe right = Mute notifications
```

### Long Press
```
Message:
Long press = Show menu (Copy, Report, Block)

Group:
Long press = Show options (Mute, Leave, Info)
```
