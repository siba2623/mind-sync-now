## 🚀 Quick Start Commands

### Build for Mobile
```bash
npm run mobile:build
```

### Open Native IDEs
```bash
# Android Studio
npm run mobile:android

# Xcode (macOS only)  
npm run mobile:ios
```

### Run on Devices
```bash
# Android device
npm run mobile:run-android

# iOS device
npm run mobile:run-ios
```
